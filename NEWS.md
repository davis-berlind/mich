# mich 0.0.1

This is the initial release version of [`{mich}`](../index.html). This release 
introduces:

- Main package function `mich()`.
- Summary and plotting S3 methods for the `mich.fit` class.
- A `NEWS.md` file to track changes to the package.
- Weighted prior functions `log_mean_prior()`, `log_var_prior()`, and 
  `log_meanvar_prior()`.
- Datasets [`well_log`](../reference/well_log.html), 
  [`ion_channel`](../reference/ion_channel.html), and 
  [`daily_steps`](../reference/daily_steps.html)
- New vignettes:
    - [Introduction](../articles/mich.html)
    - [Daily Steps](../articles/step_count.html).
