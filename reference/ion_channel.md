# Ion Channel Voltage

Ionic current recordings from a porin in the outer membrane of a
bacterial cell using the voltage clamp technique. Data have been
processed so that they include every 11th observation.

## Usage

``` r
ion_channel
```

## Format

### `ion_channel`

A data frame with 2,956 rows and 1 columns:

- y:

  Ionic current recordings
